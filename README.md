 Java Student Management System

 📌 Project Description

This is a simple console-based Student Management System built using Java.
The program allows users to add student details, view all students, and delete a student from the system.

 🛠 Technologies Used

* Java
* VS Code

 ▶️ How to Run the Project

1. Clone the repository.
2. Open the project folder in VS Code.
3. Compile the program using:
   javac StudentManagement.java
4. Run the program using:
   java StudentManagement

 🎯 Features

* Add new students (ID, Name, Marks)
* View all students
* Delete student by ID
* Menu-driven console interface

 📚 Concepts Used

* Classes and Objects
* ArrayList
* Scanner class
* Loops
* Conditional statements
* Basic Object-Oriented Programming (OOP)
